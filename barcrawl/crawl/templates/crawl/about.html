About Barcrawl

Barcrawl started as a class project to try and implement a traveling salesmsan approximation algorithm. Originally it was only a Python console application, but now it's been developed to be available as a web application so that others could enjoy the results!


How to Use

  1. Input your starting address
    (Your personal data is not saved and will disappear once your session ends)
  2. Input the cities you want to travel to
    (will automatically give one bar per city)

    OR

    Leave the field blank, press "Skip", and input the number of bars in your city that you want to visit
  3. Get your bar crawl route
